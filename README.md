# WhiteNoise 白噪音混合播放器

一款极简风格的白噪音混合播放器，帮助你专注、放松和入眠。

## 功能特性

- 🎵 **66+ 高品质音效** - 涵盖雨声、雷雨、自然环境、水声、火焰、城市环境等
- 🎛️ **实时混音** - 同时播放多个音效，独立控制每个音轨音量
- 🌙 **深色禅意设计** - 极简界面，减少视觉干扰
- 📱 **响应式布局** - 支持桌面和移动设备
- 🔄 **无缝循环** - 音频自动循环播放

## 快速开始

### 环境要求

- Python 3.7+
- Flask

### 安装依赖

```bash
pip install flask pyyaml
```

### 启动服务

```bash
cd /home/patrickzeng/dev/mine/WhiteNoise
python server.py
```

### 访问应用

打开浏览器访问: **http://localhost:5000**

## 使用方法

1. **选择音效** - 点击音效卡片将其添加到混音面板
2. **调节音量** - 拖动每个音轨的滑块调节单独音量
3. **播放/暂停** - 点击蓝色播放按钮开始播放
4. **主音量** - 拖动底部主音量滑块调节总音量
5. **移除音效** - 点击音轨右侧的 × 按钮移除
6. **清除全部** - 点击右上角清除按钮移除所有音效
